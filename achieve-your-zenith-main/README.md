# FitTrack Health Coach

## Description
A personalized virtual fitness coaching platform that helps users track workouts, monitor nutrition, set fitness goals, and access tailored wellness plans. The platform features a user-friendly dashboard with real-time progress tracking and curated workout recommendations.

## Features
* Personalized dashboard with fitness stats (steps, calories, active time, heart rate)
* Curated workout recommendations with video guides
* Weekly goal progress tracking (workout sessions, active minutes, calories burned)
* Nutrition tracker with protein/carb suggestions and full meal plans
* Recent activity timeline for workout history
* Responsive design for mobile and desktop use
* User authentication (login/signup) and profile management
* Virtual chatbot support (planned feature)

## Installation
```bash
# Install dependencies
npm install

# Or use bun (alternative package manager)
bun install
```

## Usage
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linting
npm run lint
```

## Technologies Used
* Core: React 18, TypeScript
* Build Tool: Vite 5
* Styling: Tailwind CSS, ShadCN UI, Tailwind Merge
* State Management: React Hook Form, Zod (validation)
* UI Components: Radix UI, Lucide React Icons
* Routing: React Router DOM
* Data Handling: TanStack React Query
* Additional: Recharts (data visualization), Sonner (toasts), Vaul (drawers)
